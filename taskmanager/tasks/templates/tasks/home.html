Task Management API

Bienvenue sur l'API de gestion des tâches. Voici les endpoints disponibles:

Authentification

POST /api/token/ - Obtenir un token JWT

POST /api/token/refresh/ - Rafraîchir un token JWT

Utilisateurs

POST /api/users/ - Créer un nouvel utilisateur

GET /api/users/ - Voir son profil

Tâches

GET /api/tasks/ - Liste des tâches

POST /api/tasks/ - Créer une tâche

GET /api/tasks/{id}/ - Détails d'une tâche

PUT /api/tasks/{id}/ - Mettre à jour une tâche

DELETE /api/tasks/{id}/ - Supprimer une tâche

PATCH /api/tasks/{id}/mark_complete/ - Marquer comme terminée

PATCH /api/tasks/{id}/mark_incomplete/ - Marquer comme non terminée

Filtres et tri

GET /api/tasks/?status=pending - Filtrer par statut

GET /api/tasks/?priority=high - Filtrer par priorité

GET /api/tasks/?ordering=due_date - Trier par date d'échéance

Pour plus d'informations, consultez la documentation complète de l'API.
